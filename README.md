## Getting Started

1. Clone the repository
```bash
git clone https://github.com/yourusername/signforme-ai.git
```

2. Install dependencies
```bash
cd signforme-ai
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
